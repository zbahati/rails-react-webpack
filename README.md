## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.


### Prerequisites

In order to run this project you need:

-Commands: npm install

### Setup
* git clone https://github.com/zbahati/rails-react-webpack.git
* cd rails-react-webpack
* git checkout development

### Install
-Command: install Ruby
-Command: gem install rails
- [OFFICIAL](https://rubyinstaller.org/downloads/)



### Run in Browser to see the application
- Browser:
 run yarn build --watch in one terminal
 run rails s in other terminal

### run test
-Command: Run through IRB / Vs Code
-Command: rails s
